# CRPaste Privacy Policy / 隐私政策

Last updated: 2026-03-04

This Privacy Policy describes how CRPaste handles user data.  
本隐私政策说明 CRPaste 如何处理用户数据。

## 1. Data Collection / 数据收集

CRPaste does not collect personal account information for analytics in the current version.  
CRPaste 当前版本不采集用于分析的个人账号信息。

The app may process the following local data only for core functionality:  
应用仅为实现核心功能在本地处理以下数据：
- Clipboard content you copy (text, image, file references, etc.) / 你复制的剪贴板内容（文本、图片、文件引用等）
- App preferences (hotkeys, language, history limit) / 应用设置（快捷键、语言、历史条数）
- Local favorites data / 本地收藏数据

## 2. Data Usage / 数据用途

Data is used solely to provide app features such as clipboard history, search, paste, OCR/QR tools, and favorites management.  
数据仅用于提供剪贴板历史、搜索、粘贴、OCR/二维码、收藏管理等功能。

## 3. Data Storage / 数据存储

- Data is stored locally on your Mac by default.  
- In the current App Store release, favorites sync is local-only by default.  

- 数据默认保存在你的 Mac 本地。  
- 当前 App Store 版本默认使用本地收藏数据模式。

## 4. Data Sharing / 数据共享

CRPaste does not sell user data.  
CRPaste 不出售用户数据。

CRPaste does not automatically upload clipboard history or favorites content to external servers.  
CRPaste 不会自动将剪贴板历史或收藏内容上传到外部服务器。

## 5. Permissions / 权限说明

### Accessibility / 辅助功能
- Purpose: optional auto-paste (simulate `Command + V`) after user action.  
- If denied, copy still works; only auto-paste is unavailable.

- 用途：用户主动选择后执行可选自动粘贴（模拟 `Command + V`）。  
- 未授权时仍可复制，仅自动粘贴不可用。

### Screen Recording / 屏幕录制
- Purpose: screenshot capture and image tools (OCR / QR).  
- If denied, screenshot-related features are unavailable.

- 用途：截图与图片能力（OCR/二维码）。  
- 未授权时截图相关功能不可用。

## 6. Diagnostics Export / 诊断报告导出

If you choose `Export Local Diagnostics`, the app generates a local text file on your device.  
如果你主动使用“导出本地诊断报告”，应用会在本机生成文本文件。

The file is not uploaded automatically.  
该文件不会被自动上传。

## 7. Your Choices / 用户选择

You can:
- Clear clipboard history in app menu / 在应用菜单中清空历史  
- Remove or manage favorites / 管理或删除收藏  
- Revoke permissions in macOS System Settings / 在 macOS 系统设置中撤销权限
