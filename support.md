## FAQ

### 1) Why do I need Accessibility permission? / 为什么需要“辅助功能”权限？
- Used only for optional auto-paste (simulate `Command + V`) after you pick an item.
- If not granted, CRPaste still copies content to system clipboard.

### 2) Why do I need Screen Recording permission? / 为什么需要“屏幕录制”权限？
- Used only for screenshot capture and image tools (OCR / QR).
- Core clipboard history and paste are still available without this permission.

### 3) Does CRPaste upload my clipboard data? / 会上传剪贴板数据吗？
- No telemetry upload in current version.
- Data is stored locally by default.

### 4) How to restore Pro subscription? / 如何恢复 Pro 订阅？
- Open app menu -> `Restore Purchases`.
- Ensure the same Apple ID used for the original purchase.

## Troubleshooting / 故障排查

1. Quit and relaunch CRPaste / 退出后重启 CRPaste  
2. Verify permissions in System Settings / 在系统设置中检查权限  
3. Use `Export Local Diagnostics` in app menu and send report to support / 导出本地诊断报告并发送给支持邮箱
